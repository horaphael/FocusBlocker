## Description pour le Chrome Web Store

**Titre** : Focus Blocker - Pomodoro & Site Blocker

**Description courte (132 caractères max)** :
Bloque les sites distrayants et gère ton temps avec le Pomodoro Timer intégré. Reste concentré, travaille mieux !

**Description détaillée** :

Focus Blocker t'aide à rester concentré en bloquant les sites qui te distraient (YouTube, TikTok, Facebook...) et en gérant ton temps avec la technique Pomodoro.

FONCTIONNALITÉS :

**Timer Pomodoro intégré**
- Durées personnalisables (travail + pause)
- Fenêtre séparée qui reste visible
- Notifications à chaque fin de session
- Statistiques de productivité

**Blocage intelligent**
- Mode manuel : Active/désactive quand tu veux
- Mode Pomodoro : Blocage automatique pendant le travail, libre pendant les pauses
- Sites bloqués : YouTube, TikTok, Facebook (personnalisable)

**Interface moderne**
- Design élégant avec dégradés
- Logo personnalisé
- Page de blocage motivante
- Animations fluides

**Suivi de productivité**
- Nombre de sessions Pomodoro complétées
- Total de minutes de focus
- Progression visible

**Respect de la vie privée**
- Toutes les données restent locales
- Aucune collecte d'informations personnelles
- Open source

**Parfait pour :**
- Étudiants en période d'examens
- Développeurs qui codent
- Freelances qui travaillent à domicile
- Tous ceux qui veulent améliorer leur productivité

## Justification des permissions

### storage
Pour sauvegarder tes paramètres Pomodoro et statistiques localement.

### declarativeNetRequest
Pour bloquer l'accès aux sites distrayants configurés.

### notifications  
Pour t'alerter à la fin de chaque session Pomodoro.

### host_permissions: <all_urls>
Nécessaire pour rediriger vers la page de blocage quand tu visites un site bloqué.
